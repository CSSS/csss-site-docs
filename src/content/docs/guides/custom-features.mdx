Some custom features were added to the standard framework. These will need to be maintained by the W3 Committee.

## Sidebar
There is a custom sidebar at [`src/sidebar.ts`](https://github.com/CSSS/csss-site-docs/blob/master/src/sidebar.ts).

* You can assign both a `slug` and an `items` field to create article links that also have indented children.
The `slug` value indicates the article that should be referenced.
The `items` value is an array of the articles that should appear underneath the parent article.
